Example Description

This example describes how to use i2c by using mbed api

1.Connect LOG-UART connector to PC
  
2.Connect 
  - I2C1 SDA (PF_3) to I2C2 SDA (PE_8) pin and also connect to a pull-up resistor,
  - I2C1 SCL (PF_2) to I2C2 SCL (PE_7) pin and also connect to a pull-up resistor.
  
3.Run the main function.

4.Get the Master and Slave Data.
